Power Showdown Instructions

Welcome to the Power Showdown Game

This game is based on a Power Showdown League called

National Showdown League, (NShL)

To play, in the file

power-showdown.htm

Is where the game is at.

Getting Set Up

First, you will choose the Away Team, Home Team, and their records.

Optional, you can also set up the quarter, and the timer.

Game rules and how to play

This game is played on a table, similar to Air Hockey.

Both ends of the table have a hole that have a net to catch the ball in.

When you make a goal, thats worth 2PTS

When you make a penalty, thats worth 1PT for the opposite team.

Normally a game ends with one team ending with 11+PTS, if you are using the timer, you can go until it hits 0

You also can go through the quarters with the timer.

Beneith the point buttons, is a generator.

If the number is 1, 2, 4, or 5, the ball didn't make if anywhere

If its a 3, thats a penalty

If its a 6, its a goal.

When trying to tell who has the ball, start the person with the ball who is away, then hit the ball for the person home.

Enjoy Power Showdown!
